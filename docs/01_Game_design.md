# 01. Game Design Document (GDD)

## 1. 專案概述
- **目標**：重製經典 Flash 遊戲《Flipside》(Nitrome) 的核心體驗。
- **類型**：基於物理的 2D 橫向捲軸競速平台遊戲 (Physics-based Platform Racer)。
- **核心體驗**：流暢的速度感、對抗重力的吸附機制、雙人競爭。

## 2. 核心機制 (Core Mechanics)

### A. 動量與吸附 (Momentum & Adhesion)
- **速度閾值 (Velocity Threshold)**：
  - `Speed > Threshold`：車輛產生「下壓力」，吸附於表面（無視全域重力，法線=重力方向）。
  - `Speed < Threshold`：車輛失去抓地力，受全域重力影響（掉落）。
- **地形互動**：
  - 玩家可在天花板、牆壁、360度迴圈上行駛。
  - 障礙物（如三角錐）的主要作用是「減速」，導致玩家因速度不足而掉落。

### B. 車輛控制
- **地面模式**：左右鍵控制輪胎轉速（摩擦力驅動）。
- **空中模式**：左右鍵控制車身力矩（Torque），調整落地姿態。
- **雙面行駛**：車輛為中心對稱設計，無「翻車」概念，輪子接觸即視為落地。

## 3. 遊戲流程
- **勝利條件**：到達終點線。
- **失敗懲罰**：掉出地圖或撞毀後在最近 Checkpoint 重生（時間懲罰）。
